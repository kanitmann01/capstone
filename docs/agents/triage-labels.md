# Triage Label Vocabulary

| Role | Label | Purpose |
|------|-------|---------|
| Needs evaluation | `needs-triage` | Maintainer needs to evaluate the issue |
| Waiting on reporter | `needs-info` | Blocked waiting on additional information from reporter |
| Ready for agent | `ready-for-agent` | Fully specified, can be picked up by an AFK agent with no human context |
| Ready for human | `ready-for-human` | Needs human implementation (too complex/vague for autonomous agent) |
| Won't fix | `wontfix` | Will not be actioned |
